JS Memory Game

Do you think you have a good memory? Let's test it!


Rules
  1. Press the START button
  2. The computer will select the first color
  3. Repeat the color the Computer selected
  4. The Computer will duplicate the first color and add one
  5. Repeat the same two colors the Computer selected, in the same order
  6. Continue the game while you can repeat the color the Computer selected
  7. The game is finished when you lose or when you win by successfully complete 20 rounds
  8. Press the START button again if you wish to play again

Good Luck!
